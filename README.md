# Customer Churn Prediction

This project predicts whether a telecom customer is likely to churn based on customer information and service details. It uses multiple machine learning models to train and evaluate the data, along with hyperparameter tuning to improve model performance.

The project also includes a FastAPI backend for serving predictions and a simple Streamlit web application that allows users to enter customer details and get predictions through an interactive interface.

### Tech Stack

- Python
- Scikit-learn
- Pandas
- NumPy
- FastAPI
- Streamlit
- Docker
